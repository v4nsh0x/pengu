Installation

Pengu is compiled natively via Go. To get started, clone the repository and build the binary.

git clone https://github.com/v4nsh0x/pengu.git
cd pengu
go build -o pengu .

Add the binary to your system PATH to run `pengu` from anywhere.

Quick Start

Create a file called hello.pen and write your first Pengu program:

say "Hello, Pengu! 🐧"

Run it using the CLI:

pengu hello.pen

CLI Usage

Pengu provides a robust command-line interface for running and compiling your scripts.

🚀

Run Scripts

pengu run <file.pen>
💻

Interactive REPL

pengu repl
📦

Compile to Binary

pengu build <file> -o <out>
🌐

Package Manager

pengu install <module>

Variables

Declare variables using the store keyword. Pengu is dynamically typed, so you don't need to specify types.

store name = "Vansh"
store age = 20
store isHacker = true

// Reassign values directly
age = 21

Functions & Closures

Define functions using the fn keyword. Pengu supports first-class functions and closures!

fn add(x, y) {
    return x + y
}

say add(10, 5) // 15

// Functions as values
store multiply = fn(a, b) {
    return a * b
}
say multiply(4, 5) // 20

Control Flow

Use when / otherwise for conditionals and repeat for loops.

Conditionals

store score = 85

when score >= 90 {
    say "A"
} otherwise {
    when score >= 80 {
        say "B"
    } otherwise {
        say "C"
    }
}

Loops

// While loop
store i = 0
repeat i < 5 {
    say i
    i = i + 1
}

// Iterator loop
store nums = [1, 2, 3]
repeat n in nums {
    say n
}

Array Methods

Pengu provides blazing fast, natively built-in array methods for transforming data instantly.

Method Description Returns
.push(item) Adds an item to the end of the array New length
.pop() Removes the last item Removed item
.map(fn) Transforms each element New array
.filter(fn) Keeps elements matching the condition New array
.reduce(fn, init) Accumulates values Final value
.find(fn) Finds first matching element Item or null
.includes(val) Checks if array contains value Boolean 
store nums = [1, 2, 3, 4, 5]

store evens = nums.filter(fn(x) { return x % 2 == 0 })
say evens // [2, 4]

store doubled = evens.map(fn(x) { return x * 2 })
say doubled // [4, 8]

HTTP Module

Native Built-in

Pengu is built for modern networking and cybersecurity tooling. The native http module lets you make blazing fast REST API calls.

Import it anywhere using use http.

use http

// Basic GET request
store res = http.get("https://jsonplaceholder.typicode.com/todos/1")
say res.status

// Automatically parse JSON
store data = res.json()
say "Title: " + data["title"]

// POST request with body and headers
store postRes = http.post(
    "https://api.example.com/data",
    { name: "Pengu", secure: true },
    {
        headers: { "Authorization": "Bearer token123" },
        timeout: 5
    }
)

The response object exposes status, text, body, headers, and the json() method.
