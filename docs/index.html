Installation

Choose the method that works best for you. No runtime or dependencies needed — Pengu ships as a single binary.

Download Pre-built Binary

Grab the latest release for your platform from GitHub Releases. No Go installation required.

Platform Architecture Download
🪟 Windows x64 pengu-windows-amd64.exe
🐧 Linux x64 pengu-linux-amd64
🐧 Linux ARM64 pengu-linux-arm64
🍎 macOS Apple Silicon pengu-darwin-arm64
🍎 macOS Intel pengu-darwin-amd64

After downloading, rename the binary to pengu (or pengu.exe on Windows) and add it to your system PATH.

Linux / macOS quick setup:

# Example for Linux x64
curl -L -o pengu https://github.com/v4nsh0x/pengu/releases/latest/download/pengu-linux-amd64
chmod +x pengu
sudo mv pengu /usr/local/bin/

Windows quick setup (PowerShell):

# Create a directory and download the executable
New-Item -ItemType Directory -Force -Path "C:\pengu"
Invoke-WebRequest -Uri "https://github.com/v4nsh0x/pengu/releases/latest/download/pengu-windows-amd64.exe" -OutFile "C:\pengu\pengu.exe"

# Add C:\pengu to your system PATH
[System.Environment]::SetEnvironmentVariable("PATH", $env:PATH + ";C:\pengu", "User")

Build from Source

Requires Go 1.21+.

git clone https://github.com/v4nsh0x/pengu.git
cd pengu
go build -o pengu .
🔄

Self-Updating

Once installed, Pengu can update itself to the latest version automatically:

pengu update

Quick Start

Create a file called hello.pen and write your first Pengu program:

say "Hello, Pengu! 🐧"

Run it using the CLI:

pengu hello.pen

Pengu provides a robust command-line interface for running and compiling your scripts.

Run Scripts

Interactive REPL

📦

Compile to Binary

🌐

Package Manager

Variables

Declare variables using the store keyword. Pengu is dynamically typed, so you don't need to specify types.

store name = "Vansh"
store age = 20
store isHacker = true

// Reassign values directly
age = 21

Functions & Closures

Define functions using the fn keyword. Pengu supports first-class functions and closures!

fn add(x, y) {
    return x + y
}

say add(10, 5) // 15

// Functions as values
store multiply = fn(a, b) {
    return a * b
}
say multiply(4, 5) // 20

Control Flow

Use when / otherwise for conditionals and repeat for loops.

Conditionals

store score = 85

when score >= 90 {
    say "A"
} otherwise {
    when score >= 80 {
        say "B"
    } otherwise {
        say "C"
    }
}

Loops

// While loop
store i = 0
repeat i < 5 {
    say i
    i = i + 1
}

// Iterator loop
store nums = [1, 2, 3]
repeat n in nums {
    say n
}

store evens = nums.filter(fn(x) { return x % 2 == 0 })

HTTP Module

Native Built-in

Pengu is built for modern networking and cybersecurity tooling. The native http module lets you make blazing fast REST API calls.

Import it anywhere using use http.

use http

store res = http.get("https://jsonplaceholder.typicode.com/todos/1")
say "Status: " + toString(res.status)
say "Data: " + res.json()["title"]
Method Description Returns
http.get(url, options?) Perform an HTTP GET request Response Object
http.post(url, body?, options?) Perform an HTTP POST request Response Object
http.put(url, body?, options?) Perform an HTTP PUT request Response Object
http.delete(url, options?) Perform an HTTP DELETE request Response Object

Response Object Properties

Property / Method Description
status HTTP status code (integer)
text / body Raw string response body
headers Object containing response headers
json() Method that parses the body as JSON

The response object exposes status, text, body, headers, and the json() method.

Net Module

Native Built-in

The net module handles raw TCP networking, concurrent port scanning, and comprehensive DNS lookups.

use net
store ips = net.dns_lookup("google.com")
store ports = net.scan(ips[0], [80, 443, 8080], 2) // 2s timeout
repeat port in ports {
    store status = ports[port]
    when status == "open" {
        say "Port " + port + " is OPEN"
    }
}
Method Description Returns
net.scan(host, [ports], timeout?) Concurrently scans an array of ports Object { port: "open"|"closed" }
net.scan_range(host, start, end, timeout?) Concurrently scans a range of ports Array of open ports
net.connect(host, port, timeout?) Creates a raw TCP connection Connection Object
net.dns_lookup(domain) Resolves A records (IPv4) Array of Strings
net.reverse_dns(ip) Performs a reverse DNS lookup Array of Strings
net.lookup_mx(domain) Resolves MX (mail) records Array of Objects
net.lookup_ns(domain) Resolves NS (nameserver) records Array of Strings
net.lookup_txt(domain) Resolves TXT records Array of Strings

Recon Module

External (requires install)

The recon module is a high-level external module written in Pengu that automates web and network reconnaissance.

First install it using pengu install recon, then use it in your scripts:

use recon
store info = fingerprint("https://google.com")
say info["server"]
Method Description Returns
fingerprint(url) Analyzes HTTP headers to identify server technologies Object (server, powered_by, technologies)
check_headers(url) Audits security headers (HSTS, CSP, X-Frame-Options) Array of Objects
check_sensitive_files(url) Scans for exposed files (.env, .git, backups) Array of Objects
dir_scan(url, [paths]) Checks if an array of paths exists on the server Array of Objects
grab_banner(host, port) Connects via TCP and reads the service banner String
dns_recon(domain) Performs A, MX, NS, and TXT lookups at once Object

Why I Built This

I started building Pengu because I wanted to understand how programming languages actually work internally instead of just using them.

Most people write code every day without thinking about what happens between typing code and seeing it run. I wanted to learn that process from the ground up, so I built everything myself — the lexer, parser, AST, interpreter, runtime, and CLI.

But beyond just learning, I built it to solve a personal frustration. I seriously love Golang so much — it's fast, compiled, and powerful. However, it often feels too complex and verbose for simple scripting tasks or automation. I found myself needing something with the simple, clean syntax of Python, but backed by the execution speed and single-binary deployment of Go.

Pengu is the result: a language that is incredibly easy to write, but runs at near-native speeds under the hood. The codebase is kept simple and readable on purpose so it's easy to explore and learn from, especially for people interested in interpreters, compilers, or systems programming.
